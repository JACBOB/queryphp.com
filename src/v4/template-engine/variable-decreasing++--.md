我们有时候需要对一个变量进行**递增**、**递减**操作，这个时候我们可以很方便使用下面的代码标签。当然，递增一个变量，有多种方法，我们介绍三种方法。

# 1：使用 code ~：
~~~
{~$nValue++}   
~~~

模板编译后的结果：
~~~
<?php $nValue++; ?>
~~~

# 2：使用 node php 标签：
~~~
<php>$nValue--;</php>
~~~

模板编译后的结果：
~~~
<?php $nValue--; ?>
~~~

# 3：使用 PHP 自身
~~~
<?php $nValue--; ?>
~~~
  
模板编译后的结果：
~~~
<?php $nValue--; ?>
~~~
