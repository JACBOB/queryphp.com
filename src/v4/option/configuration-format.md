QueryPHP 框架中所有配置文件的定义格式均采用返回 PHP 数组的方式，格式为： 
~~~
return [ 
        'url_router_on' => true
];
~~~

配置参数**区分大小写**（配置参数大小写为不同的配置，我们推荐使用小写），所以下面的配置等效：
还可以在配置文件中可以使用多维数组来配置更多的信息，例如：
~~~
return [ 
        'test_option' => [ 
                'hello_option' => [ 
                        'world' => 5,
                        'me' => '你好' 
                ] 
        ] 
];  
~~~ 
 
系统目前最多支持 **N 维数组**的配置级别，每个项目配置文件除了定义 QueryPHP 系统内置的配置参数之外，开发人员可以在里面添加项目需要的一些配置参数，用于自己的应用。
